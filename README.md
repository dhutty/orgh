Orgh
===============================

version number: 0.1.0
author: Duncan Hutty

Overview
--------

A helper for Github/GHE orgs' repositories

Installation / Usage
--------------------

To install use pip:

    $ pip install orgh


Or clone the repo:

    $ git clone https://github.com/dhutty/orgh.git
    $ python setup.py install
